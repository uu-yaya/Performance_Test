# 桌宠性能采集操作指南：从环境准备开始

这份指南用于边测试边复制命令。BongoCat 当前按你的实际情况使用进程名 `bongo-cat`。

## 0. 测试前默认信息

```powershell
$ProjectDir = "D:\desktop_pet"
$ExcelPath = "D:\desktop_pet\桌宠类产品性能测试记录表.xlsx"
$Collector = "D:\desktop_pet\tools\collect_desktop_pet_perf.ps1"
$Importer = "D:\desktop_pet\tools\import_perf_results.py"
$Python = "C:\Users\zoeuuliu\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe"
```

BongoCat 默认：

```powershell
$ProductId = "P001"
$ProductName = "BongoCat"
$ProcessName = "bongo-cat"
```

如果你不确定真实进程名，先启动产品，再运行：

```powershell
Get-Process | Sort-Object ProcessName | Select-Object ProcessName,Id,CPU,WorkingSet64 | Out-GridView
```

如果 BongoCat 没启动，可以尝试：

```powershell
Start-Process "C:\Program Files\BongoCat\bongo-cat.exe"
```

正式采集前一定先确认进程能被找到：

```powershell
Get-Process -Name bongo-cat -ErrorAction SilentlyContinue |
  Select-Object ProcessName,Id,CPU,@{Name="MemoryMB";Expression={[math]::Round($_.WorkingSet64/1MB,1)}} |
  Format-Table -AutoSize
```

如果没有输出，不要开始采集；先确认 BongoCat 是否已经启动，以及进程名是否正确。

## 1. 重启电脑并等待

手动重启电脑，重启后等待 5 分钟，让系统后台任务稳定。

记录当前时间：

```powershell
Get-Date
```

---

## 2. 关闭干扰应用

建议关闭浏览器、微信/QQ/Discord、Steam 下载、云盘同步、游戏启动器、录屏/直播、IDE、其他桌宠或壁纸软件。

查看 CPU 高占用进程：

```powershell
Get-Process |
  Sort-Object CPU -Descending |
  Select-Object -First 20 ProcessName,Id,CPU,@{Name="MemoryMB";Expression={[math]::Round($_.WorkingSet64/1MB,1)}} |
  Format-Table -AutoSize
```

查看内存高占用进程：

```powershell
Get-Process |
  Sort-Object WorkingSet64 -Descending |
  Select-Object -First 20 ProcessName,Id,@{Name="MemoryMB";Expression={[math]::Round($_.WorkingSet64/1MB,1)}} |
  Format-Table -AutoSize
```

## 3. 记录测试环境

系统版本：

```powershell
Get-ComputerInfo |
  Select-Object WindowsProductName,WindowsVersion,OsHardwareAbstractionLayer,OsTotalVisibleMemorySize |
  Format-List
```

CPU：

```powershell
Get-CimInstance Win32_Processor |
  Select-Object Name,NumberOfCores,NumberOfLogicalProcessors,MaxClockSpeed |
  Format-List
```

GPU：

```powershell
Get-CimInstance Win32_VideoController |
  Select-Object Name,AdapterRAM,DriverVersion,VideoModeDescription |
  Format-List
```

分辨率和刷新率：

```powershell
Get-CimInstance Win32_VideoController |
  Select-Object Name,CurrentHorizontalResolution,CurrentVerticalResolution,CurrentRefreshRate |
  Format-Table -AutoSize
```

电源模式：

```powershell
powercfg /GETACTIVESCHEME
```

## 6. 启动并确认 BongoCat

启动 BongoCat 后等待 30-60 秒，再确认进程：

```powershell
Get-Process -Name bongo-cat -ErrorAction SilentlyContinue |
  Select-Object ProcessName,Id,CPU,@{Name="MemoryMB";Expression={[math]::Round($_.WorkingSet64/1MB,1)}} |
  Format-Table -AutoSize
```

如果没有输出，说明进程名没匹配上，采集结果会全是 0。
